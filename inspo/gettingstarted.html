How it works

Setup in seconds.
Not sessions.

01 —

Find your tool

Browse the registry by category — AI coding, video generation, audio, models, SDKs. Every tool is indexed with version, OS support, and prerequisites.

gettingstarted.sh/browse/ai-coding
02 —

Run one command

Copy a single curl command. The setup script handles dependencies, environment variables, and initial configuration — verified by the community.

curl gettingstarted.sh/[tool] | sh
03 —

Docs that stay current

Each tool page links to the canonical setup guide with copy-paste commands, common errors, and version-pinned instructions maintained by contributors who use it daily.

gettingstarted.sh/cursor/docs
04 —

Contribute a tool

Missing a tool? Submit a PR. The schema is simple: a manifest file + setup script + docs page. Verified tools get indexed within 48 hours.

gs add --tool=[name] --category=[tag]
